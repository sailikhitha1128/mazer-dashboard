PROJECT DESCRIPTION

This project is a simple admin dashboard created for Task 3 of the frontend assessment.
It demonstrates UI customization, data handling, multi-page structure, and interactive elements such as search and charts.

MAIN FEATURES

1. Dashboard Page

* Displays total users, total sales, and orders
* Shows top 5 users based on highest sales
* Includes a sales bar chart
* Search bar filters users immediately

2. Users Page

* Shows full list of users with names, roles, and sales
* Search bar filters user list by name and role
* Users are automatically sorted by highest sales first

3. Data Management

* All values are loaded through JavaScript
* Users and stats are stored in a structured JS object

4. Multi-Page Structure

* index.html (Dashboard)
* users.html (User List)
